# step1:设计默认首页
## 将界面主要划分为三块：1:header 2:main  3:footer
## 将每个划分内的内容进行填充，并在style标签内设计样式，使用flex来进行布局。
### 1 header：包括主标题和副标题
### 2 main：包含四张快捷建议卡片
### 3 footer:主要是提问栏和两个功能按钮。

### 遇到的一些问题：
 1：在点击模式切换时只是会闪一下，点击后还是暗夜模式。原因：button默认是submit，在点击时会触发表单提交，会应该调整为普通button
2：切换模式后，卡片和按钮的背景颜色并没有变化。 因为在style中对于CSS选择器的书写有要求，需要在body的class名与后面的容器名之间留一个空格，这样才是后代选择器，否则意义就变为了在一个标签中同时拥有这两个类才会执行。

# step2:按键功能实现效果

## 1：主题切换：初始为深色模式，点击月亮按钮后会转换为浅色模式，图标变为太阳。
### 过程：在CSS中，对于与颜色有关的项，都会定义两套样式，一个是默认，另一个是在body下新增的一个light类，当body有这个类时，就会执行，把前面的样式覆盖掉，否则不会执行，保留默认样式。在javaScript中，通过按钮触发动作来决定light类的增删，实现对于颜色深浅的控制。

###  问题：当点击刷新按钮时并不会保留之前的模式，都会重置为默认深色模式。
